## 7.1 设计规范

### 7.1.1 尺寸单位
- 字体大小：`fp` (Font Pixel)
- 间距、边距：`vp` (Virtual Pixel)
- 屏幕宽度：`100%` 或具体 `vp` 值

### 7.1.2 通用间距
| 间距类型 | 值 | 用途 |
|----------|-----|------|
| 极小间距 | 4vp | 卡片内元素间距 |
| 小间距 | 8vp | 相关元素间距 |
| 中间距 | 16vp | 卡片内边距 |
| 大间距 | 24vp | 区块间距 |
| 超大间距 | 32vp | 页面级间距 |

### 7.1.3 字体规范
| 用途 | 大小 | 字重 |
|------|------|------|
| 大标题 | 24fp | 600 |
| 中标题 | 18fp | 600 |
| 小标题 | 16fp | 700 |
| 正文 | 14fp | 400 |
| 辅助文字 | 12fp | 400 |

### 7.1.4 颜色规范
| 用途 | 颜色名称 | 色值 |
|------|----------|------|
| 主题色 | color_theme | #FF6B6B |
| 未选中 | color_unselected | #999999 |
| 列表内容 | color_todo_list_content | #666666 |
| 背景 | color_background | #F5F5F5 |

## 7.3 主页布局

```
┌─────────────────────────────┐
│ ← 任务列表          [+]     │ ← 标题栏 (56vp)
├─────────────────────────────┤
│ [筛选▼] [排序▼]             │ ← 筛选栏 (40vp)
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 学习高数第三章    [待办] │ │
│ │ 番茄:25分钟  休息:5分钟  │ │
│ │ 已完成: 3/8个番茄        │ │
│ │      [ 开始专注 ]       │ │ ← 按钮居中 (40%宽度)
│ └─────────────────────────┘ │ ← 卡片圆角 8vp
│     ↑ 8vp                   │
│ ┌─────────────────────────┐ │
│ │ 每日学习英语    [计划]  │ │
│ │ 截止: 2026-12-31        │ │
│ │ 今日: 2/5个  累计: 127个│ │
│ │      [ 开始专注 ]       │ │
│ └─────────────────────────┘ │
│                             │
│      (更多任务...)          │
│                             │
└─────────────────────────────┘
│ [任务列表] [统计] [我的]    │ ← Tab栏 (80vp)
└─────────────────────────────┘
```

**布局要点**：
- 任务卡片宽度 96%
- 卡片左右边距 2%
- 卡片上下间距 4vp
- 卡片阴影：radius 4, offsetY 2

## 7.4 计时器页布局

```
┌─────────────────────────────┐
│                             │
│        [待办模式]            │ ← 任务标签 (16vp)
│                             │
│      ╭──────────────╮       │
│      │              │       │
│      │              │       │
│      │     25:00    │       │ ← 计时器 (50fp)
│      │              │       │
│      │              │       │
│      ╰──────────────╯       │ ← 环形进度条 (80%宽度)
│       专注模式              │ ← 状态文字 (18vp)
│                             │
│    学习高数第三章            │ ← 任务名称 (20fp, 600字重)
│                             │
│                             │
│  [ 暂停 ]    [ 结束 ]       │ ← 操作按钮 (各40%宽度)
│                             │
└─────────────────────────────┘
```

**布局要点**：
- 环形进度条：strokeWidth 24
- 按钮高度：40vp
- 按钮间距：justifyContent SpaceAround
- 底部边距：40vp

**结束弹窗布局**：
```
┌─────────────────────────────┐

│ ║  │ [继续番茄钟]        │  ║
│ ║  │ [结束并记录]        │  ║
│ ║  │ [结束不记录]        │  ║

```

## 7.7 Tab栏布局

```
┌─────────────────────────────┐
│                             │
│                             │
│         (页面内容)           │
│                             │
│                             │
├─────────────────────────────┤
│ ┌─────┐ ┌─────┐ ┌─────┐    │
│ │ [图] │ │ [图] │ │ [图] │    │ ← 图标 (24x24vp)
│ │任务  │ │统计  │ │我的  │    │ ← 文字 (12fp)
│ └─────┘ └─────┘ └─────┘    │
│   选中色    未选中色         │
└─────────────────────────────┘
```

**Tab样式**：
- 高度：80vp
- 图标大小：24x24vp
- 文字大小：12fp
- 选中颜色：color_theme
- 未选中颜色：color_unselected
- 图标文字间距：4vp
