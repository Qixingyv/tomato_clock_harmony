# 任务管理

> 版本：v1.0
> 创建日期：2026-03-18
> 状态：需求确认中

---

## 4.1 概述

任务管理功能负责任务的创建、编辑、查看、筛选、排序等操作，是用户管理番茄钟任务的核心入口。

---

## 4.2 任务列表

### 4.2.1 列表展示
- 显示所有"进行中"状态的任务
- 每个任务以卡片形式展示
- 长按可拖拽调整优先级

### 4.2.2 任务卡片内容

**待办模式卡片**
```
┌────────────────────────────┐
│ 学习高数第三章     [待办]  │
├────────────────────────────┤
│ 番茄时长: 25分钟            │
│ 休息时长: 5分钟             │
│ 已完成: 3/8个番茄           │
├────────────────────────────┤
│      [ 开始专注 ]           │
└────────────────────────────┘
```

**计划模式卡片**
```
┌────────────────────────────┐
│ 每日学习英语     [计划]    │
│ 截止: 2026-12-31  剩余:288天│
├────────────────────────────┤
│ 番茄时长: 25分钟            │
│ 休息时长: 5分钟             │
│ 今日: 2/5个  累计: 127个    │
├────────────────────────────┤
│      [ 开始专注 ]           │
└────────────────────────────┘
```

### 4.2.3 卡片交互
- **点击卡片**：进入任务编辑页面
- **点击开始按钮**：进入计时器页面

## 4.3 任务创建

### 4.3.1 创建入口
- 任务列表顶部：点击 "+" 按钮
- 底部：可添加浮动创建按钮

### 4.3.2 创建表单

| 字段 | 类型 | 待办模式必填 | 计划模式必填 | 默认值 |
|------|------|--------------|--------------|--------|
| 任务名称 | 文本输入 | ✅ 是 | ✅ 是 | - |
| 任务模式 | 单选（待办/计划） | ✅ 是 | ✅ 是 | 待办 |
| 番茄时长 | 数字输入 | ✅ 是 | ✅ 是 | 全局默认(25) |
| 休息时长 | 数字输入 | ✅ 是 | ✅ 是 | 全局默认(5) |
| 目标番茄数 | 数字输入 | ✅ 是 | ✅ 是 | 1（待办）/ 5（计划/每天） |
| 截止日期 | 日期选择 | ❌ 否 | ✅ 是 | - |

**字段说明：**
- **待办模式**：目标番茄数 = 任务总量，完成后即结束
- **计划模式**：
  - 目标番茄数 = 每天需要完成的数量
  - 截止日期 = 计划坚持到的日期
  - 累计总目标 = 每天 × 天数（系统自动计算）

### 4.3.3 表单验证

**通用验证（所有模式）：**

- 任务名称不能为空
- 番茄时长必须大于0
- 休息时长必须大于0

**待办模式专属验证：**
- 目标番茄数必须大于0
- 提示信息："请设置目标任务需要完成几个番茄钟"

**计划模式专属验证：**
- 每日目标番茄数必须大于0
- 截止日期必须晚于今天
- 提示信息："请设置每天要完成几个番茄钟" / "请选择计划截止日期"

**实时验证规则：**
- 用户切换任务模式时，自动显示/隐藏相关字段
- 切换到计划模式时，截止日期字段显示并标记为必填
- 提交时根据当前选择的模式进行对应验证

## 4.4 任务编辑

### 4.4.1 编辑入口
- 点击任务卡片进入

### 4.4.2 可编辑字段
- 任务名称
- 番茄时长
- 休息时长
- 目标番茄数
- 截止日期

### 4.4.3 不可编辑字段
- 任务模式（创建后锁定）
- 已完成番茄数

### 4.4.4 编辑操作
- **保存**：更新任务信息
- **取消**：放弃修改，返回上一页
- **删除**：取消任务（两种模式统一，需二次确认）

**取消任务确认弹窗**（两种模式通用）
```
┌─────────────────────────────┐
│ 确认取消此任务？             │
│                             │
│ 取消后任务将不再显示在列表中，│
│ 但可在历史记录中查看和恢复。 │
│                             │
│      [返回]    [确认取消]    │
└─────────────────────────────┘
```

**操作说明：**
- 两种模式都使用"取消"操作，状态变为"已取消（5）"
- 已取消任务移入历史记录
- 可通过筛选"已取消"状态查看
- 支持恢复操作（将状态改回"进行中"）

## 4.5 筛选与排序

### 4.5.1 筛选条件

| 筛选项 | 选项 | 说明 |
|--------|------|------|
| 任务状态 | 全部/进行中/已完成/已取消/已过期 | 按状态筛选 |
| 任务模式 | 全部/待办/计划 | 按模式筛选 |

### 4.5.2 排序方式

| 排序项 | 说明 |
|--------|------|
| 创建时间 | 默认排序，新任务在前 |
| 优先级 | 拖拽后的顺序 |
| 截止日期 | 计划模式按截止日期排序，截至日期越早的，在越前面 |
| 完成进度 | 按完成百分比排序 |

## 4.6 任务状态

### 4.6.1 状态定义

**待办模式状态**

| 状态码 | 名称 | 说明 | 显示位置 |
|--------|------|------|----------|
| 1 | 进行中 | 任务未完成 | 任务列表 |
| 2 | 已完成 | 任务已达成目标 | 历史记录 |
| 5 | 已取消 | 用户主动取消任务 | 历史记录 |

**计划模式状态**

| 状态码 | 名称 | 说明 | 显示位置 |
|--------|------|------|----------|
| 1 | 进行中 | 任务未完成 | 任务列表 |
| 2 | 已完成 | 任务已达成目标 | 历史记录 |
| 4 | 已过期 | 超过截止日期 | 历史记录 |
| 5 | 已取消 | 用户中途取消任务 | 历史记录（可恢复） |

### 4.6.2 状态流转

**待办模式状态流转**

```
[创建] → 进行中(1)
            ↓
    ┌───────┴───────────┐
    ↓                   ↓
 达成目标            用户取消
    ↓                   ↓
已完成(2)          已取消(5)
    ↑                   ↑
    └───────┬───────────┘
            ↓
        历史记录
```

**计划模式状态流转**

```
[创建] → 进行中(1)
            ↓
    ┌───────┴─────────────────┐
    ↓                          ↓
 达成目标                   用户取消
    ↓                          ↓
已完成(2)                  已取消(5)
    ↑                          ↑
    │                     可恢复
  超期
    ↓
已过期(4)
```

**历史记录管理**：
- 已完成任务统一显示在历史记录中
- 可通过时间筛选（今日/本周/本月/更早）管理
- MVP不单独实现"归档"功能

## 4.7 优先级管理

### 4.7.1 调整方式
- 长按任务卡片
- 拖拽到目标位置
- 释放完成调整

### 4.7.2 优先级存储
- 每个任务记录 `priority` 字段
- 默认按创建时间排序
- 拖拽后更新优先级

## 4.8 历史记录

### 4.8.1 查看入口
历史记录集成在**统计页面**中，通过以下方式访问：
- 底部Tab点击"统计"
- 统计页面顶部切换到"历史记录"子Tab

### 4.8.2 页面结构

```
统计页面
├── 子Tab导航: [概览] [历史记录]
│
└── 【历史记录Tab】
    ├── 顶部筛选栏
    │   ├── 状态筛选: [全部] [已完成] [已取消] [已过期]
    │   └── 时间筛选: [全部时间] [本周] [本月] [更早]
    │
    └── 任务列表（只读卡片）
        ├── 卡片内容: 任务名称、完成时间、完成番茄数、状态标签
        └── 卡片操作: 查看详情、恢复（已取消状态）、删除
```

### 4.8.3 历史记录卡片

**已完成任务卡片**
```
┌────────────────────────────┐
│ 每日学习英语        [计划]  │
│ 完成于: 2026-03-15          │
│ 累计完成: 500个番茄 ✅      │
│ 坚持天数: 100天             │
├────────────────────────────┤
│      [查看详情]             │
└────────────────────────────┘
```

**已取消任务卡片**
```
┌────────────────────────────┐
│ 学习高数第三章      [待办]  │
│ 取消于: 2026-03-18          │
│ 已完成: 3/8个番茄 ⚠️        │
├────────────────────────────┤
│   [恢复任务]  [删除记录]    │
└────────────────────────────┘
```

### 4.8.4 筛选功能

**状态筛选**
| 选项 | 说明 |
|------|------|
| 全部 | 显示所有历史任务 |
| 已完成 | 只显示达成目标的任务 |
| 已取消 | 只显示用户取消的任务 |
| 已过期 | 只显示计划模式过期任务 |

**时间筛选**
| 选项 | 说明 |
|------|------|
| 全部时间 | 显示所有历史记录 |
| 本周 | 完成/取消时间在本周内 |
| 本月 | 完成/取消时间在本月内 |
| 更早 | 完成/取消时间在更早 |

### 4.8.5 历史操作

| 操作 | 说明 | 适用状态 |
|------|------|----------|
| 查看详情 | 查看任务完整信息和完成历史 | 全部 |
| 恢复任务 | 将任务状态改回"进行中"，重新显示在任务列表 | 已取消 |
| 删除记录 | 永久删除任务记录（需二次确认） | 全部 |

### 4.8.6 删除确认弹窗

```
┌─────────────────────────────┐
│ 确认删除此记录？             │
│                             │
│ 删除后将无法恢复，请谨慎操作。│
│                             │
│      [返回]    [确认删除]    │
└─────────────────────────────┘
```
