# 计时器功能

> 版本：v1.0
> 创建日期：2026-03-18
> 状态：需求确认中

---

## 3.1 概述

计时器是番茄钟应用的核心功能，提供专注模式和休息模式的倒计时，支持暂停、继续、提前结束等操作。

---

## 3.2 基础计时

### 3.2.1 专注模式
- **时长**：根据任务设置的番茄时长（默认25分钟）
- **显示**：倒计时 MM:SS 格式
- **进度**：环形进度条显示剩余时间比例
- **状态**：显示 "专注模式" 或 "暂停"

### 3.2.2 休息模式
- **时长**：根据任务设置的休息时长（默认5分钟）
- **显示**：倒计时 MM:SS 格式
- **进度**：环形进度条显示剩余时间比例
- **状态**：显示 "休息模式" 或 "休息暂停"

---

## 3.3 计时器操作

### 3.3.1 操作按钮

| 操作 | 专注模式 | 休息模式 | 说明 |
|------|----------|----------|------|
| 开始 | - | - | 启动计时（从任务列表进入时自动开始） |
| 暂停 | 显示 | 显示 | 暂停当前计时 |
| 继续 | 显示 | 显示 | 恢复暂停的计时 |
| 跳过休息 | - | 显示 | 跳过休息，直接进入下一个专注 |
| 结束 | 显示 | 显示 | 提前结束当前计时 |

### 3.3.2 按钮布局

```
┌─────────────────────────┐
│                         │
│     [环形进度条]         │
│       25:00             │
│     专注模式             │
│                         │
│   学习高数第三章         │
│                         │
│  [ 暂停 ]  [ 结束 ]     │
└─────────────────────────┘
```

## 3.4 计时器状态流转

### 3.4.1 正常流转

```
[任务开始]
    ↓
[专注模式 25分钟]
    ↓
[时间到] → 自动记录番茄 → [休息模式 5分钟]
    ↓                              ↓
还有番茄?                        [时间到]
    ↓                              ↓
   是                               否
    ↓                              ↓
[下一个专注] ─────────────────→ [返回任务列表]
```

### 3.4.2 提前结束流程

```
[专注模式中]
    ↓
点击 [结束]
    ↓
┌─────────────────────────┐
│ 确认结束专注模式？       │
│                         │
│ [继续番茄钟]             │
│ [结束并记录]             │
│ [结束不记录]             │
└─────────────────────────┘
```

### 3.4.3 跳过休息流程

```
[休息模式中]
    ↓
点击 [跳过休息]
    ↓
┌─────────────────────────┐
│ 确认跳过休息？           │
│                         │
│ [跳过] → [下一个专注]    │
│ [继续休息]              │
└─────────────────────────┘
```

## 3.5 结束处理

### 3.5.1 专注模式结束

用户可选择：

| 选项 | 行为 | 数据变化 |
|------|------|----------|
| 继续番茄钟 | 取消结束，恢复计时 | 无 |
| 结束并记录 | 记录1个番茄，返回列表 | finished_tomato_count +1 |
| 结束不记录 | 不记录，直接返回 | 无 |

### 3.5.2 休息模式结束

用户可选择：

| 选项 | 行为 |
|------|------|
| 继续 | 恢复休息计时 |
| 结束 | 记录当前番茄，返回列表 |

## 3.7 通知提醒

### 3.7.1 通知时机
- 专注时间结束
- 休息时间结束

### 3.7.2 通知方式
- 系统通知栏通知
- 可选声音提醒
- 可选震动提醒

### 3.7.3 通知内容

**专注时间结束通知：**
```
专注时间结束！
你完成了一个番茄钟，休息一下吧。

[退出番茄钟] [开始休息]
```

**休息时间结束通知：**
```
休息时间结束！
准备好开始下一个番茄钟了吗？

[退出番茄钟] [开始专注]
```

**按钮说明：**

| 按钮 | 行为 |
|------|------|
| 退出番茄钟 | 返回任务列表，保存当前进度 |
| 开始休息 | 进入休息模式倒计时 |
| 开始专注 | 进入下一个专注模式倒计时 |

## 3.8 技术实现

### 3.8.1 计时器组件
- 使用 `TextTimer` 组件实现倒计时
- 使用 `Progress` 组件实现环形进度条
- 使用 `TextTimerController` 控制计时器

### 3.8.2 状态管理
```typescript
@State currentTomatoStatus: TaskStatus = TaskStatus.Working;
@State isPause: boolean = false;
@State progressValue: number = 0;
@State progressTotal: number = 0;
```

### 3.8.3 关键方法
```typescript
// 开始专注模式
startWorkingMode()

// 开始休息模式
startRestMode()

// 暂停计时
pauseTimer()

// 继续计时
resumeTimer()

// 结束计时
endTimer()
```

## 3.9 边缘场景处理

### 3.9.1 应用退到后台
- **MVP版本**：继续计时，时间到后发送通知
- **完整版**：继续计时，结束后发通知，支持应用存活优化

### 3.9.2 应用被杀死
- **MVP版本**：不处理，重新开始
- **完整版**：保存状态，重启后恢复

### 3.9.3 屏幕关闭
- 继续计时
- 亮屏后恢复显示
