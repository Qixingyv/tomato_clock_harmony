# 统计功能

> 版本：v1.0
> 创建日期：2026-03-18
> 状态：需求确认中

---

## 5.1 概述

统计功能帮助用户了解自己的专注习惯，提供两个核心视图：
- **概览**：今日、本周、本月三个时间维度的数据统计
- **历史记录**：已完成的任务列表，支持筛选和查看详情

用户通过顶部子Tab切换两个视图。

---

## 5.2 时间维度

### 5.2.1 今日统计
- 统计范围：今天 00:00 - 当前时间
- 更新频率：实时更新
- 默认显示

### 5.2.2 本周统计
- 统计范围：本周一 - 当前时间
- 更新频率：实时更新
- 可切换查看

### 5.2.3 本月统计
- 统计范围：本月1号 - 当前时间
- 更新频率：实时更新
- 可切换查看

## 5.3 统计指标

### 5.3.1 核心指标

| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 完成番茄数 | 完成的番茄钟总数 | count(finished_tomato_count) |
| 专注时长 | 总专注时间 | sum(tomato_minutes * finished_tomato_count) |
| 任务完成率 | 已完成任务占比 | completed_tasks / total_tasks * 100% |
| 平均每日番茄 | 日均完成番茄数 | total_tomatos / days |

### 5.3.2 辅助指标

| 指标 | 说明 | 适用场景 |
|------|------|----------|
| 最专注任务 | 完成番茄数最多的任务 | 了解专注重点 |
| 连续打卡天数 | 连续完成每日目标的天数 | 计划模式 |
| 最长专注时长 | 单次最长的专注时间 | 了解专注能力 |
| 累计完成番茄数 | 从创建到当前的累计完成数 | 计划模式，展示长期坚持成果 |

### 5.3.3 计划模式累计数据展示

对于计划模式的任务，统计页面会额外展示：
- **历史累计番茄数**：从任务创建到当前的总完成番茄数
- **坚持天数**：从创建到当前的天数
- **平均每日完成**：累计番茄数 ÷ 坚持天数

**示例：**
```
┌─────────────────────────────┐
│ 计划任务详情                 │
├─────────────────────────────┤
│ 任务：每日学习英语            │
│ 创建时间：2026-01-01         │
│ 坚持天数：76天               │
├─────────────────────────────┤
│ 历史累计    380 个番茄       │
│ 平均每日    5.0 个番茄       │
│ 今日完成    5/5 个           │
└─────────────────────────────┘
```

**筛选选项：**
- 状态筛选：全部 / 已完成 / 已取消 / 已过期
- 时间筛选：全部时间 / 本周 / 本月 / 更早

**任务卡片操作：**
- 已完成任务：查看详情
- 已取消任务：恢复任务、删除记录
- 已过期任务：查看详情、删除记录

## 5.6 数据计算

### 5.6.1 今日统计
```typescript
interface TodayStats {
  date: Date;           // 今天日期
  completedTomatoes: number;  // 完成番茄数
  focusMinutes: number;       // 专注时长（分钟）
  completedTasks: number;     // 已完成任务数
  totalTasks: number;         // 总任务数
  completionRate: number;     // 完成率
}
```

### 5.6.2 本周统计
```typescript
interface WeekStats {
  startDate: Date;      // 周一日期
  endDate: Date;        // 周日日期
  completedTomatoes: number;
  focusMinutes: number;
  avgDailyTomatoes: number;   // 平均每日番茄数
  taskCompletionRate: number;
  dailyData: DailyData[];     // 每日数据数组
}
```

### 5.6.3 本月统计
```typescript
interface MonthStats {
  year: number;
  month: number;
  completedTomatoes: number;
  focusMinutes: number;
  avgDailyTomatoes: number;
  taskCompletionRate: number;
  dailyData: DailyData[];
}
```
